# Artic Store

Интернет-магазин одежды, обуви и аксессуаров — фронтенд-часть. Промежуточная, но самостоятельно работающая сборка: главная, каталог с фильтрами и страница товара полностью свёрстаны и интерактивны.

## Технологии

- **Vite** — сборка и dev-сервер
- **SCSS** + нативные CSS-переменные (единая дизайн-система токенов: цвета, отступы, типографика)
- **Vanilla JS** (ES-модули, делегирование событий) — без тяжёлых фреймворков
- **Handlebars** — переиспользуемые партиалы разметки
- **Swiper** — слайдеры и карусели
- **GLightbox** — полноэкранный просмотр фото товара
- Шрифты self-hosted (Inter, Noto Sans), изображения в WebP

## Что готово

- **Главная** — hero-слайдер, карусели товаров, промо-секции, блог
- **Каталог** — фильтры (бренд / размер / цвет / цена), сортировка, пагинация, человекопонятные URL (`/catalog/{пол}/{группа}/{категория}`), боковая навигация по категориям
- **Страница товара** — галерея с миниатюрами и полноэкранным просмотром, переключение цвета и размера, табы «Описание / Доставка и возврат», блок «С этим товаром выбирают», уведомление при добавлении в корзину
- **Избранное** — добавление в избранное со счётчиком (сохраняется в браузере)
- **Адаптив** — десктоп и мобильные брейкпоинты 1440 / 1024 / 768 / 429

Данные временно отдаются из мок-слоя (`src/js/api/mockApi.js`) — структура готова к подключению реального бэкенда.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`.

## Сборка

```bash
npm run build     # продакшн-сборка в dist/
npm run preview   # локальный предпросмотр сборки
```

## Структура

```
src/
  assets/        изображения, иконки (спрайт)
  scss/          стили: токены, базовые, компоненты, секции, страницы
  js/            модули, компоненты, утилиты, мок-API
  partials/      Handlebars-партиалы (шапка, футер, секции)
index.html       главная
catalog.html     каталог
product.html     страница товара
```
